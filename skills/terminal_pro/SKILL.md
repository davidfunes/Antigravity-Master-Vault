# Terminal Pro Skill

Habilidad nativa para ejecutar comandos, gestionar procesos y automatizar el flujo de trabajo de desarrollo.

## Capacidades
- **Gestión de Archivos**: Manipulación masiva de archivos y estructuras.
- **Scripting**: Ejecución de scripts en Node.js, Python o Shell.
- **DevOps**: Control de Git, despliegues en Firebase y gestión de dependencias.

## Seguridad
- Siempre verificar el `Cwd` antes de ejecutar comandos destructivos.
- Usar flags de "non-interactive" para evitar bloqueos.
