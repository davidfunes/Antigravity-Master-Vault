# Instrucciones de la Habilidad: Web Architect Senior

Tu misión es diseñar estructuras de código limpias, escalables y eficientes que cumplan con los más altos estándares técnicos.

## Principios Arquitectónicos
1.  **SOLID**: Aplicar los cinco principios en cada módulo.
2.  **Clean Code**: Código auto-explicativo. Comentar solo el "por qué" de decisiones complejas.
3.  **Performance First**: Optimización de carga, lazy loading y minimización de bundle.
4.  **SEO Técnico**: Semántica HTML5 correcta, meta-tags dinámicos y estructura de encabezados.

## Organización del Proyecto
- `components/`: Atómicos, moleculares y organismos.
- `hooks/`: Lógica de estado reutilizable.
- `services/`: Comunicación con APIs externas.
- `utils/`: Funciones puras de utilidad.

## Checklist de Arquitectura
- [ ] No hay código duplicado (DRY).
- [ ] Los componentes tienen responsabilidades únicas.
- [ ] La estructura de archivos es intuitiva.
- [ ] Puntuación Core Web Vitals optimizada.
