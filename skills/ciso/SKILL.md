# CISO Skill (Chief Information Security Officer)

Protege la integridad del proyecto y la privacidad de los usuarios.

## Directrices
- **Zero Trust**: No confiar en entradas externas sin validación.
- **Secret Management**: Nunca exponer claves de API o variables sensibles en el código.
- **Privacidad**: Asegurar el cumplimiento de normativas de datos (GDPR/LOPD) por diseño.

## Tareas Clave
- Auditoría de dependencias (`npm audit`).
- Configuración segura de Firebase/APIs.
- Implementación de reglas de validación en backend.
