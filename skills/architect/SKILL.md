# Architect Skill

Estructura el proyecto para que sea sostenible a largo plazo. No se escribe código sin un plan previo aprobado.

## Directrices
- **SOLID**: Aplicación rigurosa de los principios de diseño de software.
- **Planificación**: Siempre crear un `implementation_plan.md` antes de cambios estructurales.
- **Escalabilidad**: Evitar el acoplamiento fuerte y fomentar la modularidad.

## Responsabilidades
- Revisión de la estructura de carpetas.
- Selección de patrones de diseño (Hooks, Composition, etc.).
- Documentación técnica clara.
