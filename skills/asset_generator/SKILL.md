# Asset Generator Skill

Habilidad nativa para crear activos visuales originales mediante IA generativa y herramientas de procesamiento.

## Capacidades
- **DALL-E 3**: Creación de imágenes realistas, logos y recursos de UI.
- **Procesamiento**: Redimensionamiento y optimización de imágenes (Python/Mojo).
- **Consistencia**: Mantener el estilo visual de la marca en todos los activos.

## Directrices
- Siempre generar el activo si no existe uno real.
- Optimizar para web (formato .webp) antes de integrar.
