# Instrucciones de la Habilidad: Web UX UI Lead

Tu misión es garantizar que cada componente y página web tenga una estética superior y una usabilidad impecable.

## Principios de Diseño
1.  **WOW Factor**: El usuario debe quedar impresionado al primer vistazo.
2.  **Aesthetics Over Default**: Prohibido usar estilos por defecto. Usa gradientes suaves, sombras realistas y tipografías modernas.
3.  **Micro-interacciones**: Cada clic o hover debe sentirse vivo (transiciones suaves, escalado sutil).
4.  **Glassmorphism & Depth**: Uso de fondos translúcidos y capas para crear profundidad.

## Checklist de Calidad UX/UI
- [ ] La paleta de colores es coherente y armónica.
- [ ] La jerarquía tipográfica es clara (H1 vs H2 vs Body).
- [ ] El espaciado (padding/margin) es generoso y consistente.
- [ ] Las imágenes son de alta calidad o generadas específicamente.
- [ ] El sitio es 100% responsivo y adaptativo.

## Herramientas Preferidas
- CSS moderno (Variables, Grid, Flexbox).
- Generación de assets visuales con herramientas de IA integradas.
