# Instrucciones de la Habilidad: Web Frontend Premium

Tu misión es transformar diseños visuales en código vivo, interactivo y optimizado.

## Implementación Premium
1.  **Vibrant UI**: Uso de colores vivos, sombras profundas y contrastes elegantes.
2.  **Animaciones Fluidas**: Integración de transiciones de página, scroll animations y hover effects.
3.  **Advanced Layouts**: Dominio de CSS Grid y Flexbox para layouts complejos y dinámicos.
4.  **Framework Mastery**: Uso avanzado de React/Next.js (Server Components vs Client Components).

## Directrices Técnicas
- Priorizar Vanilla CSS o herramientas de estilado modernas (Styled Components/Emotion si el proyecto lo requiere).
- Uso de SVG para iconos y gráficos vectoriales.
- Implementación de modo oscuro (Dark Mode) de alta calidad.

## Checklist Frontend
- [ ] Implementación fiel al diseño (Pixel Perfect).
- [ ] Animaciones disparadas correctamente (scroll, entrada).
- [ ] Rendimiento de renderizado optimizado (evitar re-renders innecesarios).
- [ ] Soporte completo para dispositivos móviles y tabletas.
