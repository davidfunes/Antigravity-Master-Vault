# Browsing Expert Skill

Habilidad nativa para operar navegadores web, extraer información de documentación y validar despliegues.

## Capacidades
- **Investigación**: Búsqueda de soluciones a errores complejos y APIs.
- **Auditoría**: Uso de herramientas de desarrollador para inspeccionar el DOM y el tráfico de red.
- **Validación**: Pruebas de flujo de usuario punta a punta.

## Protocolo
1. Definir el objetivo de la navegación.
2. Capturar screenshots para evidencia visual.
3. Resumir hallazgos con enlaces a fuentes oficiales.
