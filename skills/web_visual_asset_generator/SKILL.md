# Instrucciones de la Habilidad: Web Visual Asset Generator

Tu misión es dotar a la web de una identidad visual única a través de imágenes, iconos y vídeos de alta calidad.

## Estética Visual
1.  **Brand Consistency**: Asegurar que cada imagen generada siga la paleta de colores y el estilo del proyecto.
2.  **Asset Optimization**: Uso de formatos modernos (WebP, AVIF, SVG) para asegurar cargas ultrarrápidas.
3.  **Unique Art**: Generación de arte original (IA Generativa) para evitar fotos de stock genéricas.
4.  **Audio-Visual Sync**: (Especialmente para David Funes Music) Optimización de carátulas y visualizadores de audio.

## Checklist de Activos Visuales
- [ ] Las imágenes están comprimidas sin pérdida perceptible de calidad.
- [ ] Todos los iconos son vectoriales (SVG).
- [ ] La relación de aspecto es correcta para cada dispositivo.
- [ ] El video (si existe) tiene un poster de carga y está optimizado para streaming.

## Herramientas Preferidas
- Generadores de imágenes por IA (DALL-E, Midjourney, etc.).
- Scripts de procesamiento de imagen (Python/Sharp).
- Editores de vectores.
